# Data Licenses and Source Notes

This repository contains derived benchmark prompts, normalized model outputs, metrics, and analysis artifacts used for scholarly reproducibility.

## Source Data Notes

- **BBC text classification prompts:** derived from BBC news classification data used for research benchmarking. Original source terms should be respected when reusing the source text.
- **OULAD structured decision-support prompts:** derived from Open University Learning Analytics Dataset-style records. Original OULAD terms and citation requirements should be respected.
- **UltraChat / WildChat-derived open-ended reasoning prompts:** used as public conversational/instructional examples for benchmark construction. Original dataset licenses and terms should be checked before redistribution or reuse beyond scholarly verification.

## Generated Outputs

The model outputs are generated responses from API-based experiments. They are included to support reproducibility of the manuscript's ECS, EVR, and EDI calculations.

## Recommended Reuse

Use these files for peer review, replication, and scholarly auditing of the reported results. For any downstream dataset redistribution, verify the original source licenses for each component dataset.
